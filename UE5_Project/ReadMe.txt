操作方法
　・wasdで移動
　・マウス操作で視点移動
　・1 2 3 4キーで武器装備
　・釣り竿装備状態でFキーで釣り(釣り場にいるときのみ)
　・shiftキーで回避
　・武器装備状態でマウス左クリックで攻撃
　・spaceキーでジャンプ
